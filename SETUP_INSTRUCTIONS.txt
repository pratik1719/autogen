================================================================================
TROUBLESHOOTING
================================================================================

Problem: "No module named 'google.generativeai'"
Solution: pip install --upgrade google-generativeai

Problem: "GEMINI_API_KEY not found"
Solution: Make sure .env file exists and has your API key

Problem: "CSV file not found"
Solution: Check the file path, use forward slashes: data/file.csv

Problem: "Rate limit exceeded"
Solution: Wait a minute and try again (free tier has limits)

Problem: "Analysis takes too long"
Solution: 
- Use a smaller dataset (< 50,000 rows)
- Or sample your data first: df.sample(n=10000).to_csv(...)

Problem: "Plots look weird"
Solution: This is normal - the system generates plots for any data structure.
Some datasets just have unusual distributions.

Problem: "LLM generates generic insights"
Solution: This happens occasionally. The system has fallbacks, but you can
re-run the analysis to get different insights.
